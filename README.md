## Installation

1. Import the module

```javascript
import { NgxSwingModule } from 'ngx-swing';
```

2. Add the module to your NgModule

```javascript
@NgModule({
	...
    imports: [
    	....
        NgxSwingModule
        ....
    ]
})
```

3. Enjoy with components

```html
<div swingStack>
	<div swingCard>Card1</div>
    <div swingCard>Card2</div>
    <div swingCard>Card3</div>
    <div swingCard>Card4</div>
</div>
```
